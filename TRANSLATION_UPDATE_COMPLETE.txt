================================================================================
HOW IT WORKS
================================================================================

The Laravel application uses JSON translation files for multi-language support:

1. User clicks language selector (English, Arabic, French, German)
2. LanguageController sets locale in session: $request->session()->put('locale', $locale)
3. LocaleMiddleware ensures locale persists: App::setLocale(session()->get('locale'))
4. Templates use __() function: {{ __('key') }} to retrieve translations
5. JavaScript uses window.__() function for dynamic translations
6. Laravel automatically loads the correct JSON file based on current locale

When Arabic is selected:
- All template strings via __() load from /lang/ar.json
- Application displays all 1,019 keys in Arabic
- 100% coverage of UI elements, buttons, messages, and dialogs
